# Riddhi Siddhi - Crystal Jewelry Shop

A mobile-first e-commerce website for crystal jewelry.

## Features

- **Home page**: Hero section, top categories, welcome/contact info
- **Product listing**: Bracelet category with filter and sort
- **Bottom navigation**: Home, Search, Account, Cart
- **Floating buttons**: WhatsApp and Chat support
- **Responsive design**: Optimized for mobile, scales to larger screens

## Getting Started

1. Open `index.html` in a web browser (or use a local server).
2. Navigate between pages using the bottom nav and category links.
3. Product pages link to placeholder product detail pages.

## Project Structure

```
├── index.html          # Home page
├── products.html       # Bracelet product listing
├── product-detail.html # Single product view
├── cart.html           # Shopping cart
├── search.html         # Search / browse
├── account.html        # Account / contact
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Basic interactivity
└── README.md
```

## Customization

- **Colors**: Edit CSS variables in `:root` (primary, accent, etc.)
- **Contact**: Update phone number and WhatsApp links (+91 95880 16011)
- **Products**: Replace placeholder product images with real image URLs
- **Add backend**: Connect to a CMS or API for dynamic products
